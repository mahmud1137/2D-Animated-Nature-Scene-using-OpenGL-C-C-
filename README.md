2D Animated Nature Scene using OpenGL (C/C++)

This project is a dynamic 2D animated environment built with OpenGL and GLUT. It simulates a natural landscape featuring day-night transitions, moving clouds, flying birds, flowing river waves, and multiple animated boats.

The scene includes an interactive weather system where users can toggle rainfall in real time. When rain is activated, the environment adapts with a cloudy sky, falling rain particles, and disabled sun/moon animations, creating a realistic weather effect.

Key Features:

Day & Night cycle with animated sun and moon
Moving clouds and flying birds with wing animation
Realistic water waves using sine functions
Multiple animated boats with different speeds
Interactive rainfall system (keyboard controlled)
Dynamic sky color transitions

Controls:

Press 'r' → Start Rain
Press 's' → Stop Rain
